Git-GitHub Notlari

- git init ile klasorumuzde git'i baslatmış oluruz.
- git add . komutu ile yaptigimiz degisiklikleri working directory'den staging area'a tasiriz.
- git status ile commit öncesinde nerede oldugumuzu goruruz.
- git commit -m "V1" komutu ile degisiklikleri staging area'dan commit store'a taşırız ve bir versiyon olusturmuş oluruz.
- git commit --amend -m "V2" mesajı degistirebiliriz
- git diff ile working space'deki degisiklikleri gorebiliriz.
- git show hashcode ile de commit store'daki degisiklikleri goruruz.
